Etch-A-Sketch

Description

This browser version of the Etch-A-Sketch allows users to create rainbow pixel drawings using mouse movement and a linear grid increase button. The HTML and CSS provide the structure and styling for the Etch-A-Sketch, while the JavaScript powers the interactive elements and enables users to control the drawing.

Purpose

  • Apply DOM Manipulation knowledge to a functional web page
  • Further develop understanding of JavaScript
  • Experiment with CSS effects and decoration

Technologies Used

  • HTML
  • CSS
  • JavaScript

What I Learned

  • Efficient self-teaching techniques
  • Combining JS functions with CSS
  • Page formatting methods

What Can/Will Be Improved

  • Formatting for different screen sizes
  • More diverse color options
  • Higher quality grid size system
